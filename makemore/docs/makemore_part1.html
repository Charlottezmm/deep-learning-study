4. Negative Log Likelihood

makemore part 1 / 04
  • Look up probability of the true next char.
  • High probability gives small penalty.
  • Low probability gives large penalty.
  • Average NLL is the loss.

log_likelihood = 0.0
n = 0

for w in words:
    chs = ['.'] + list(w) + ['.']
    for ch1, ch2 in zip(chs, chs[1:]):
        ix1 = stoi[ch1]
        ix2 = stoi[ch2]
        prob = P[ix1, ix2]
        log_likelihood += torch.log(prob)
        n += 1

loss = -log_likelihood / n
